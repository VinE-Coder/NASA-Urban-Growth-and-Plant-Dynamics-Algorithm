# NASA-Urban-Growth-and-Plant-Dynamics-Algorithm
This program parses through MRLC times series data collected from satellites and develops trend graphs to portray the change in plant growth in conjunction to urban development. This provides valuable insights on how urban growth has influenced plant dynamics and biodiversity in certain areas. In addition, this project also functions connected to the AI Workflow relying on CNN infrastructure. This workflow takes an input (ground truth or satellite data), processes the image, and outputs the validated finding. The output includes the percentages of fractional impervious surfaces (concrete, etc.), grass/plant growth, and tree canopy growth.
